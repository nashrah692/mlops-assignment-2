# MLOps Assignment 2: End-to-End Pipeline

**Student Name:** Nashrah Ahmed Khan (22F-3156)
**Course:** MLOps

## Project Overview
This project implements a complete MLOps pipeline including data versioning (DVC), CI/CD (GitHub Actions), Containerization (Docker), Orchestration (Airflow), and Cloud Deployment (AWS EC2).

## Key Features
- Data Versioning: Managed using DVC with local storage.
- Model Training: Logistic Regression pipeline triggered by DVC.
- CI/CD: Automated testing via GitHub Actions (CML).
- Orchestration: Apache Airflow DAGs for workflow management.
- Deployment: FastAPI wrapper containerized with Docker and deployed on AWS EC2.
